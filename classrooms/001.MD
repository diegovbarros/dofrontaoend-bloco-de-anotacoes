# Dicas do Terminal
## Lista de comandos

1. Comando `cd` - Navega entre os diretórios

__Exemplo__: Para entrar em uma pasta
```
cd nome-da-pasta
```

__Exemplo__: Retorna um nível anterior
```
cd ..
```

2. Comando `mkdir` - Cria um diretório

__Exemplo__: Criar uma pasta
```
mkdir nome-da-pasta
```
